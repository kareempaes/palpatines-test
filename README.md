# palpatines-test

This is a small mono repo displaying some star wars information

# Server

## Running the server

To run the server, run:

```bash
npm run start
```

To run the server in development mode, run:

```bash

npm run dev
```

## Running the tests

To run the tests, run:

```bash

npm run test
```

## Running the linter

To run the linter, run:

```bash

npm run lint
```
